CSS

CSS stands for Cascading Style Sheets and it is the language used to give style to a web page. While HTML deal with the page’s structure, CSS deals with how the page looks.

A good analogy to make is to think about a web page as a building. All the walls, floors and the steel skeleton are represented by HTML. While all the painting, design and finishing are represented by CSS.

As HTML, CSS is not a programming languages for the same reasons. It does not perform any programmed task and its purpose is only to design the layout for a page.

How does CSS work?

Let’s start with an example. The following HTML code represents a heading followed by a paragraph:

<h1>This is my heading</h1>
<p>Some text written here.</p>

In order to change the heading colour to red and the paragraph font family to arial, we would write the following CSS code:

h1 {colour: red;}
p {font-family: Arial;}

Note that the coding structure of CSS is formed by a selector (ex: h1) followed by semi columns, embedding a property (ex: colour), with some value (ex: red) attached to it.

The selector refers to which HTML element the CSS code is interacting and the property is some characteristic of this element which has a value attached to it.

Another good example is the colour of this paragraph that was given the value “green” using CSS code.

These were only some examples of CSS, but in fact, the properties of elements and the values that you can give to it are numerous and it allows you to change anything that you want in your page.

Another important thing to mention is that there are three ways to apply CSS code to your HTML page:

  • Internal
  • Inline
  • External

Internal style sheets are written in the header of a .html document inside a <style> tag. It is useful if you want to apply the style to a whole single page.

Inline style sheets are written inside a tag. It is useful if you want to give style for a single element inside your page. As it already indicates which element you want to modify, the code is written a bit different and it would look like this if you wanted to change a single heading colour:

<h1 style=”color:red;”>This is my heading</h1>

Finally, external style sheets are written in a separated .css file and linked to a page by a <link> tag placed in the head of the .html document. It is useful when you have more than one page (most of the cases) and you want to apply this style to all of these pages at once.

In summary, CSS is the tool that you apply in a HTML file in order to make a web page attractive. However, there are still more that can be made to a web page, you can make it interactive with some scrip language such as JavaScript.
