Web Development

HTML

HTML stands for Hypertext Markup Language, it is the standard language to write documents that are displayed in a web browser as a web page. It allows developers to create all the structure necessary for web page, such as heading, paragraphs, links, sections and many others.

Contrary to popular belief, HTML is not a programming language since it does not have functional purposes, only structural purposes. For example, theres is no if/else statement in HTML as it has in others programming languages. When a browser reads the content of a .html document, it tells where is the heading, paragraphs, links and the browser displays them as such.

Coding in HTML is structured by tags and attributes to build a web page. For example, if you want to add a heading in a .html document, that is the tag you would use:

<h1>That is my heading</h1>

How does HTML work?

As explained, HTML code use tags inside angle brackets to indicate what is being represented in the code. In general (with few exceptions), tags must be initialised with <> and closed with </>, as the example in the section above. That is how the browser separate elements in your code.

Another important informations about tags, is that there are three tags that every .html document needs to contain:

  1. The <html></html> tag is the highest level element that encloses every HTML page.
  2. The <head></head> tag holds meta information such as the page’s title, charset and reference links.
  3. The <body></body> tag encloses all the information that appears on the page.

Apart from those three, you can find some examples below of common tags used in web pages. Note that the closing tags </> are not represented in the table below, but they are needed for all the tags inside this table with the exception of <br>.


Tag Description
<b> Indicate some text bold
<br> Add a line break
<em> Emphasise text
<h1> to <h6> Indicate sections headings
<i> Indicate italic text
<li> Open a list
<ol> Open a numbered list
<ul> Open a bulleted list
<p> Indicate a paragraph
<div> Open a section
<a> Indicate a hyperlink

In summary, HTML deals with all the structure of a web page. However, pages are much more than only plain text, they also have style and behaviour to make the user experience even better. That is the reason why .html documents usually have a CSS and a JavaScript files associated with it.
